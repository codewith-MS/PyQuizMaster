# PyQuizMaster
An interactive quiz application built using Python and Streamlit that allows users to take quizzes based on difficulty level with real-time scoring and performance tracking.

Features
- Difficulty selection (Easy / Medium / Hard)
- Randomized questions
- Score tracking
- Accuracy calculation
- Time tracking

Tech Stack
- Python
- Streamlit
- JSON (for question storage)
- Random module
- Session State management
